Features

1. Download an audio file from a youtube video link
2. Download all the songs from an entire youtube playlist
3. Download a video file from a youtube video link
4. Download all the videos from an entire youtube playlist
5. Convert an audio file
6. Convert a video file
7. Convert an image file
8. Convert a document (see Doc Types Guide below)
Audio formats: MP3, M4A, AAC, WAV, FLAC, OPUS, and OGG
Video formats: MP4, WEBM, MKV, AVI, and MOV
Image formats: PNG, JPEG, BMP, GIF, TIFF, WEBP
Doc formats: see below
Doc Types Guide

1. DOCX → PDF
Used for: job applications, web uploads
Tool: docx2pdf (Windows) or libreoffice

2. PDF → DOCX
Used for: editing locked forms or resumes
Tool: pdf2docx, or LibreOffice if available

3. EPUB → PDF
Used for: printing or reading e-books without an e-reader
Tool: pypandoc or ebook-convert from Calibre

4. DOCX → TXT or PDF → TXT
Used for: screen readers, note-taking, copy-pasting
Tool: pypandoc or PyMuPDF

5. ODT / PAGES → DOCX
Used for: compatibility with Microsoft Word
Tool: libreoffice or iCloud export







Instructions

1. Click the download button.
2. If your computer says it's an untrusted file, it's because I'm not a trusted publisher or something. Just click allow or whatever.
3. Open your downloads folder.
4. Right-click and extract zip file and click extract.
5. If it gives you trouble again, just click trust file of whatever.
6. Go back to your downloads. You should see a folder (not the zip) called Mix Tape.
7. Click into that, and there's the program! Click and drag onto your desktop.
8. Double-click the program. Give it a few seconds to load.
